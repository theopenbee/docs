---
title: 快速开始
description: 几分钟内启动并运行 OpenBee
---

## 1. 生成配置

运行交互式配置向导：

```bash
openbee config
```

向导会引导你完成以下设置：
- Claude Code CLI 路径和超时时间
- 平台凭据（飞书、钉钉、企业微信、Telegram）

配置默认保存到 `config.yaml`。你可以指定不同的输出路径：

```bash
openbee config -o /path/to/config.yaml
```

## 2. 启动服务器

```bash
openbee server
```

或作为后台守护进程运行：

```bash
openbee server -d
```

Web 控制台地址为 `http://localhost:8080`（或你配置的主机:端口）。

## 3. 创建数字员工

打开 Web 控制台，创建你的第一个数字员工：

1. 导航到**数字员工**页面
2. 点击**创建数字员工**
3. 设置名称（如 "Assistant"）和描述
4. 可选添加初始记忆以定义数字员工的角色

## 4. 发送消息

如果你已配置了某个平台（如飞书），在 IM 应用中向你的机器人发送消息。OpenBee 会将消息路由到相应的数字员工。
