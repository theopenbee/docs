## 创建飞书应用

1. 前往[飞书开放平台](https://open.feishu.cn/)
2. 创建一个新的自建应用
3. 在应用凭证页面记录 **App ID** 和 **App Secret**

## 配置权限

为应用添加以下权限：
- `im:message` — 发送和接收消息
- `im:message.group_at_msg:readonly` — 接收群内 @消息
- `im:message.p2p_msg:readonly` — 接收单聊消息
- `im:message.reactions:write_only` — 添加消息表情回复
- `im:message:send_as_bot` — 以机器人身份发送消息
- `im:resource` — 访问媒体资源

### 一键导入配置

你也可以在飞书开放平台的应用权限页面使用**批量开通**功能，将以下 JSON 粘贴导入，即可一键完成权限配置：

```json
{
  "scopes": {
    "tenant": [
      "im:message",
      "im:message.group_at_msg:readonly",
      "im:message.p2p_msg:readonly",
      "im:message.reactions:write_only",
      "im:message:send_as_bot",
      "im:resource"
    ],
    "user": []
  }
}
```

## 设置事件订阅

1. 进入应用，选择**事件与回调** → **事件配置**
2. 将订阅方式改为**长连接**
3. 点击**添加事件**，搜索并添加 `im.message.receive_v1`（IM message receive V1）事件

## 配置 OpenBee

在 `config.yaml` 中添加：

```yaml
bee:
  platforms:
    feishu:
      enabled: true
      app_id: "your-app-id"
      app_secret: "your-app-secret"
      max_media_size: 104857600  # 100MB，可选
```

## 发布应用

1. 在飞书开放平台提交应用审核
2. 审核通过后，用户即可找到并向你的机器人发送消息

## 媒体支持

飞书支持发送和接收媒体文件（图片、文档、音频、视频），最大不超过配置的 `max_media_size`（默认 100MB）。OpenBee 在可用时会使用 FFprobe/FFmpeg 进行媒体处理。
