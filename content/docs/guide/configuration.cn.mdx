OpenBee 使用 YAML 配置文件。可以通过 `openbee config` 交互式生成，也可以手动创建。

## 完整示例

```yaml
server:
  port: 8080
  host: localhost
  debug: false
  auth:
    username: admin
    password: ""
    jwt_secret: ""
    access_token_ttl: 2h
    refresh_token_ttl: 168h

database:
  path: ./data/openbee.db

bee:
  claude:
    path: claude
    timeout: 30m
  mcp:
    api_key: "your-api-key"
  platforms:
    feishu:
      enabled: false
      app_id: ""
      app_secret: ""
      max_media_size: 104857600
    dingtalk:
      enabled: false
      client_id: ""
      client_secret: ""
    wecom:
      enabled: false
      bot_id: ""
      secret: ""
      websocket_url: "wss://openws.work.weixin.qq.com"
    telegram:
      enabled: false
      token: ""
      max_media_size: 52428800
      auth_code: ""
    weixin:
      enabled: false
      token: ""
      base_url: "https://ilinkai.weixin.qq.com"
      cdn_base_url: "https://novac2c.cdn.weixin.qq.com/c2c"
      user_id: ""
      max_media_size: 104857600
  feeder:
    timeout: 5m
  media:
    ffprobe_path: ffprobe
    ffmpeg_path: ffmpeg
  message_debounce: 3s
```

## 服务器

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `server.port` | integer | `8080` | HTTP 服务器端口 |
| `server.host` | string | `localhost` | 服务器绑定地址 |
| `server.debug` | boolean | `false` | 启用调试模式（详细日志） |

## 认证

基于 JWT 的 Web UI 认证。认证是可选的——仅在设置了 `password` 时启用。

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `server.auth.username` | string | `admin` | 登录用户名 |
| `server.auth.password` | string | — | 登录密码。为空时禁用认证。 |
| `server.auth.jwt_secret` | string | — | HMAC-SHA256 签名密钥。未设置时启动时自动生成。 |
| `server.auth.access_token_ttl` | duration | `2h` | 访问令牌有效期 |
| `server.auth.refresh_token_ttl` | duration | `168h` | 刷新令牌有效期（7 天） |

## 数据库

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `database.path` | string | `./data/openbee.db` | SQLite 数据库文件路径 |

## Claude Code

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `bee.claude.path` | string | `claude` | Claude Code CLI 二进制文件路径 |
| `bee.claude.timeout` | duration | `30m` | 每次 Claude Code 调用的最大执行时间 |

## MCP

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `bee.mcp.api_key` | string | — | MCP 服务器认证 API 密钥。MCP 工具正常运行所必需。 |

## 平台

详细的平台配置请参阅[平台接入](/cn/docs/guide/platforms)。

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `bee.platforms.feishu.enabled` | boolean | `false` | 启用飞书集成 |
| `bee.platforms.feishu.app_id` | string | — | 飞书应用 ID |
| `bee.platforms.feishu.app_secret` | string | — | 飞书应用密钥 |
| `bee.platforms.feishu.max_media_size` | integer | `104857600` | 最大媒体文件大小（100MB） |
| `bee.platforms.dingtalk.enabled` | boolean | `false` | 启用钉钉集成 |
| `bee.platforms.dingtalk.client_id` | string | — | 钉钉客户端 ID |
| `bee.platforms.dingtalk.client_secret` | string | — | 钉钉客户端密钥 |
| `bee.platforms.wecom.enabled` | boolean | `false` | 启用企业微信集成 |
| `bee.platforms.wecom.bot_id` | string | — | 企业微信机器人 ID |
| `bee.platforms.wecom.secret` | string | — | 企业微信机器人密钥 |
| `bee.platforms.wecom.websocket_url` | string | `wss://openws.work.weixin.qq.com` | 企业微信 WebSocket 地址 |
| `bee.platforms.telegram.enabled` | boolean | `false` | 启用 Telegram 集成 |
| `bee.platforms.telegram.token` | string | — | Telegram 机器人令牌 |
| `bee.platforms.telegram.max_media_size` | integer | `52428800` | 最大媒体文件大小（50MB） |
| `bee.platforms.telegram.auth_code` | string | — | `/auth` 命令的验证码。为空时允许所有用户使用。 |
| `bee.platforms.weixin.enabled` | boolean | `false` | 启用微信集成 |
| `bee.platforms.weixin.token` | string | — | 微信令牌（通过扫码登录获取） |
| `bee.platforms.weixin.base_url` | string | `https://ilinkai.weixin.qq.com` | 微信 API 基础地址 |
| `bee.platforms.weixin.cdn_base_url` | string | `https://novac2c.cdn.weixin.qq.com/c2c` | 微信 CDN 基础地址 |
| `bee.platforms.weixin.user_id` | string | — | 微信用户 ID（扫码登录时自动获取） |
| `bee.platforms.weixin.max_media_size` | integer | `104857600` | 最大媒体文件大小（100MB） |

## 投喂器

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `bee.feeder.timeout` | duration | `5m` | 蜂巢（协调器）投喂超时时间 |

## 媒体

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `bee.media.ffprobe_path` | string | `ffprobe` | FFprobe 二进制文件路径 |
| `bee.media.ffmpeg_path` | string | `ffmpeg` | FFmpeg 二进制文件路径 |

## 消息处理

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `bee.message_debounce` | duration | `3s` | 将多条快速消息聚合为一条的时间窗口 |
