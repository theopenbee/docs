---
title: Web 控制台
description: 通过 Web 界面管理 OpenBee
---

Web 控制台提供了用于管理数字员工、监控任务和查看执行日志的图形界面。

## 访问控制台

在浏览器中打开 `http://localhost:8080`（或你配置的主机:端口）。控制台与 API 一起自动提供服务。

## 仪表盘

概览页面展示：
- 数字员工总数及其状态分布
- 最近的执行记录
- 系统统计信息

## 数字员工管理

- **创建数字员工** — 设置名称、描述和初始记忆
- **编辑数字员工** — 更新其配置
- **删除数字员工** — 并可选择删除其工作目录
- **查看数字员工详情** — 包括状态和执行历史

## 执行日志

- 查看所有数字员工的执行记录
- 按数字员工或会话筛选
- **实时日志流** — 实时查看执行输出
- 查看执行结果和状态

## 会话监控

- 跟踪各平台的活跃会话
- 查看每个会话中哪些数字员工有活跃上下文
- 在需要时清除会话上下文
