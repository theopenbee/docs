MCP 服务器运行在 `/mcp/sse`，使用 Server-Sent Events 传输协议。所有请求需要 `bee.mcp.api_key` 进行认证。

## 工作者管理

### list_workers

列出所有已注册的工作者。

**参数：** 无

**返回：** 工作者对象数组，包含 `id`、`name`、`description`、`memory`、`work_dir`、`status`、`created_at`、`updated_at`。

### get_worker

根据 ID 获取单个工作者。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `worker_id` | string | 是 | 工作者 UUID |

### create_worker

创建新工作者。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `name` | string | 是 | 工作者显示名称 |
| `description` | string | 否 | 工作者用途 |
| `memory` | string | 否 | 初始持久化记忆 |
| `work_dir` | string | 否 | 自定义工作目录 |

### update_worker

更新工作者属性（补丁语义）。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `worker_id` | string | 是 | 工作者 UUID |
| `name` | string | 否 | 新名称 |
| `description` | string | 否 | 新描述 |
| `memory` | string | 否 | 新记忆内容 |

### delete_worker

删除工作者。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `worker_id` | string | 是 | 工作者 UUID |
| `delete_work_dir` | boolean | 否 | 同时删除工作目录 |

## 任务管理

### create_task

为工作者创建任务。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `message_id` | string | 是 | 来源消息 ID |
| `worker_id` | string | 是 | 目标工作者 UUID |
| `instruction` | string | 是 | 任务指令 |
| `type` | string | 否 | `immediate`（默认）、`countdown` 或 `scheduled` |
| `scheduled_at` | int64 | 条件必填 | Unix 毫秒时间戳，`countdown` 类型必填 |
| `cron_expr` | string | 条件必填 | 5 字段 cron 表达式，`scheduled` 类型必填 |

### list_tasks

带筛选条件查询任务。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `message_id` | string | 否 | 按消息筛选 |
| `session_key` | string | 否 | 按会话筛选 |
| `worker_id` | string | 否 | 按工作者筛选 |
| `status` | string | 否 | 按状态筛选 |
| `type` | string | 否 | 按类型筛选 |

### cancel_task

取消待处理或已调度的任务。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `task_id` | string | 是 | 任务 UUID |

### mark_task_complete

将任务标记为已完成。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `task_id` | string | 是 | 任务 UUID |

## 消息传递

### send_message

向用户发送回复。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `message_id` | string | 是 | 要回复的原始消息 |
| `content` | string | 是 | 回复文本内容 |
| `media_path` | string | 否 | 要附加的媒体文件路径 |

## 会话管理

### clear_session

清除所有任务并重置会话上下文。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `session_key` | string | 是 | 会话标识符 |
| `force` | boolean | 否 | 即使有任务正在运行也强制清除 |

### list_session_contexts

列出会话中有活跃上下文的工作者。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `session_key` | string | 是 | 会话标识符 |

### clear_worker_session

重置特定工作者的 Claude 会话上下文。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `session_key` | string | 是 | 会话标识符 |
| `worker_id` | string | 是 | 工作者 UUID |

## 记忆

### save_memory

保存或更新记忆条目。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `scope` | string | 是 | `global` 或会话键 |
| `key` | string | 是 | 记忆键 |
| `value` | string | 是 | 记忆值 |

### get_memory

检索记忆条目。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `scope` | string | 是 | `global` 或会话键 |
| `key` | string | 否 | 指定键（省略则返回该范围内所有条目） |

### delete_memory

删除记忆条目。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `scope` | string | 是 | `global` 或会话键 |
| `key` | string | 是 | 记忆键 |

## 监控

### get_worker_status

查看工作者状态和待处理任务数量。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `worker_id` | string | 是 | 工作者 UUID |

### get_system_overview

获取系统整体统计信息。无参数。

**返回：** 按状态统计的工作者数量、按状态统计的任务总数、最近的执行记录。

### list_bee_executions

查看蜂巢协调器执行历史。

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `limit` | int | 否 | 最大返回数量（默认 10） |
